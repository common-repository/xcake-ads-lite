xCake Ads Lite allows you to add banners to your sidebar via a widget. Still in beta-testing mode (expect daily updates)

== Description ==

xCake Ads Lite is a simple plugin that allows you to ad any type of banners to your widget-friendly theme. Just install and configure it!

*Please be aware that this is a beta version, so bugs are expected. If you find any, please report them to canha at xcakeblogs.com.br with a detailed report of what you tried doing and what was the error. Beta testers will be credited on official release.*

== Installation ==

1. Upload, activate and configure.
1. To display ads, go to the Widgets page and include the `xCake Ads` module to your widgetizes area.

== Frequently Asked Questions ==

= What languages are supported? =

Currently only Portuguese (pt_BR) and English. Default is English.

= I have a question. How do I proceed? =

Head to http://wordpress.org/tags/xcake-ads-lite?forum_id=10#postform and leave a question. If you have found a bug, please report it to canha at xcakeblogs.com.br with as much info as you can (what did you try to do, what error appeared, etc).
